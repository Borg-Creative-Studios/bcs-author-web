Literata Variable Font
======================

This download contains Literata as both variable fonts and static fonts.

Literata is a variable font with these axes:
  opsz
  wght

This means all the styles are contained in these files:
  Literata-VariableFont_opsz,wght.ttf
  Literata-Italic-VariableFont_opsz,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
